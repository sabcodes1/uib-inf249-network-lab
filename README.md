# INF249 – Network Security Lab (MA1)

I denne oppgaven bygget jeg et isolert virtuelt nettverk og utførte en rekke 
praktiske sikkerhets- og nettverksøvelser.

Hva jeg gjorde:
- Designet og implementerte en nettverkstopologi med to subnett og en router i VirtualBox.
- Konfigurerte statisk ruting og aktivert IP-forwarding i Linux.
- Utførte omfattende trafikkanalyse med Wireshark (ICMP, ARP, TCP/HTTP, SSH).
- Eksperimenterte med netcat-baserte bind- og reverse shells og analyserte risikoen ved ukryptert trafikk.
- Satt opp SSH-server og beskyttet den med fail2ban.
- Implementerte og testet en lokal brannmur (ufw) med “default deny”-policy.
- Dokumenterte hvordan ulike protokoller oppfører seg på tvers av subnett.
